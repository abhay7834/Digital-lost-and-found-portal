# Digital Lost and Found Portal

## 📌 Description
This project is a web-based application that helps users report lost items and find them easily. Users can register, login, and manage lost or found items.

---

## 🚀 Features
- User Registration & Login
- Add Lost Items
- Add Found Items
- Claim Items
- Notifications System

---

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

---

## ⚙️ How to Run

### 1. Clone the repository
git clone https://github.com/abhay7834/Digital-lost-and-found-portal.git

### 2. Install dependencies
cd client
npm install

cd ../server
npm install

### 3. Run project
cd server
npm start

cd ../client
npm start

---

## 🔒 Note
Create a `.env` file in the server folder and add required environment variables.
